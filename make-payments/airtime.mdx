With the Xente API, you can send airtime to mobile numbers.
## Step 1 - Verify phone number

To ensure your transfering funds to the correct phone number, please first verify the phone number 

<CodeGroup>

```bash cURL
curl https://api.xente.co/auth/login?
-H "Authorization: Bearer YOUR_SECRET_KEY"
-X GET
```

```bash Node
const axios = require('axios');

const apiUrl = 'https://api.xente.co/verify?';

const options = {
  headers: {
    'Authorization': 'Bearer YOUR_SECRET_KEY'
  }
};

axios.get(apiUrl, options)
  .then(response => {
    // Handle the response data here
    console.log(response.data);
  })
  .catch(error => {
    // Handle any errors that occur
    console.error(error);
  });
```

## Step 3 - Receive Instant Payment Notification (IPN)

All transactions on Xente are asynchronous. Therefore, after you POST a mobile money transaction, you need to wait for a success (or failure) response using webhooks called Instant Payment Notifications (IPN).

Learn more about **[Instant Payment Notifications](/start/ipn)**.
