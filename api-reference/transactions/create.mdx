---
title: "Create Transaction"
api: "POST https://api.xente.co/transactions"
description: "This endpoint creates a new transaction in Xente."
---

### Body

<ParamField body="clientreference_id" type="string">
  A unique id from your system 
</ParamField>
<ParamField body="batch_id" type="string">
  The id to identify batch - this is useful if transaction is part of bulk transaction
</ParamField>
<ParamField body="account_number" type="string">
  Your account number
</ParamField>
<ParamField body="currency" type="string">
  currency of the transaction
</ParamField>
<ParamField body="amount" type="number">
  transaction amount
</ParamField>
<ParamField body="debitorcredit" type="number">
   1 if credit, 2 if debit. PS. A credit puts money on your account e.g mobile money collection. A debit removes money from your account. e.g. a mobile money payout
</ParamField>
<ParamField body="product" type="object">
  A product associated with transaction 
</ParamField>
<ParamField body="beneficiary" type="object">
  A beneficiary associated with the transaction e.g. team member, business, customer 
</ParamField>
### Response

<ResponseField name="success" type="number">
  Indicates whether the call was successful. 1 if successful, error code if not.
</ResponseField>

<ResponseField name="data" type="object">

The contents of the transaction

<Expandable title="Toggle object">

<ResponseField name="transaction_id" type="number">
  The unique id of the transaction 
</ResponseField>

<ResponseField name="clientreference_id" type="number">
  The id that your system uses to identify the transaction
</ResponseField>

<ResponseField name="message" type="string">
  message from Xente on status of your transaction. This can be queued, processing, successful or failed
</ResponseField>

</Expandable>

</ResponseField>

<RequestExample>

```bash Request

curl --location --request POST 'https://api.xente.co/transactions' \
--header 'Content-Type: application/json' \
--header 'Authorization: Token <token>' \
--data-raw '{
    "clientreference_id": 67890,
    "batch_id": 54321,
    "account_number": "9876543210",
    "name": "John Doe",
    "currency": "USD",
    "amount": 100.00,
    "debitorcredit": "debit",
    "product": "See product resource for details",
    "beneficiary": "See people or business for details"
}'

```

</RequestExample>
