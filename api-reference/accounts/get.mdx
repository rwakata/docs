---
title: "Get Account"
api: "GET https://api.xente.co/v2/accounts"
description: "This endpoint gets your Xente account details"
---

### Query

<ParamField query="id" type="string">
  This is the ID your account
</ParamField>

### Response

<ResponseField name="success" type="number">
  Indicates whether the call was successful. 0 if successful, error code if not.
</ResponseField>
<ResponseField name="account" type="object">

The contents of the account

<Expandable title="Toggle object">

<ResponseField name="name" type="string">
  The name of the account
</ResponseField>

<ResponseField name="description" type="string">
  The description of the account
</ResponseField>

<ResponseField name="currency" type="string">
  The currency of the account 
</ResponseField>

<ResponseField name="balance" type="string">
  The balance on the account
</ResponseField>

<ResponseField name="transactions" type="JSON or null">
  A list of transactions on the account
</ResponseField>

<ResponseField name="statement" type="JSON or null">
  An account statement showing debits and credits
</ResponseField>

</Expandable>

</ResponseField>

<RequestExample>

```bash Request
curl --location --request GET 'https://api.xente.co/v2/accounts/{id}' \
--header 'Content-Type: application/json' \
--header 'Authorization: Token <token>' \
```

</RequestExample>

<ResponseExample>

{
  "success": 0,
  "account": {

</ResponseExample>
