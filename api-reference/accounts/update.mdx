---
title: "Update Account"
api: "PUT https://api.xente.co/v2/accounts"
description: "This endpoint updates an existing account."
---

### Body

<ParamField body="name" type="string">
  name of account e.g office payments 
</ParamField>
<ParamField body="description" type="string">
  Description of account e.g. making and collecting office payments
</ParamField>
<ParamField body="currency" type="string">
  currency of the account
</ParamField>

### Response

<ResponseField name="success" type="number">
  Indicates whether the call was successful. 0 if successful, error code if not
</ResponseField>

<ResponseField name="account" type="object">

The contents of the account

<Expandable title="Toggle object">

<ResponseField name="id" type="number">
 Unique identifier of account
</ResponseField>

<ResponseField name="name" type="string">
  name of the account
</ResponseField>

<ResponseField name="name" type="string">
  description of account
</ResponseField>

<ResponseField name="balance" type="number">
  balance on account
</ResponseField>


<ResponseField name="transactions" type="JSON or null">
  transaction list on account
</ResponseField>
<ResponseField name="statement" type="JSON or null">
  account statement showing debits and credits 
</ResponseField>

</Expandable>

</ResponseField>

<RequestExample>

```bash Request
curl --location --request PUT 'https://api.xente.co/v2/accounts/{id}' \
--header 'Content-Type: application/json' \
--header 'Authorization: Token <token>' \
--data-raw '{
    "name": "name of account",
    "description": "description of account",
    "currency": "currency of account",
}'
```

</RequestExample>
