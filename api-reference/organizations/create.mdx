---
title: "Create organization"
api: "POST https://api.xente.co/v2/organizations"
description: "This endpoint creates a new organization PS. An organization needs to be verified before it can be used and this is only done through the portal"
---

### Body

<ParamField body="name" type="string">
  name of your organization e.g My company name
</ParamField>
<ParamField body="description" type="description">
  Description of organization e.g. We sell widgets
</ParamField>
<ParamField body="address" type="address">
  Physical address of organization e.g. P.O.Box 12345, Country, State
</ParamField>
<ParamField body="users" type="object">
   User object creating the organization 
</ParamField>

### Response

<ResponseField name="statusCode" type="number">
  Indicates whether the call was successful. 0 if successful, error code if not.
</ResponseField>
<ResponseField name="statusMessage" type="string">
  Status message on outcome of the API call
</ResponseField>

<ResponseField name="organization" type="object">

The contents of the organization

<Expandable title="Toggle object">

<ResponseField name="id" type="number">
  unique id of created organization e.g. 23423423
</ResponseField>

 name of your organization e.g My company name
</ResponseField>

<ResponseField name="address" type="string">
  The address of the organization.
</ResponseField>

<ResponseField name="description" type="string">
  Description of organization e.g. We sell widgets
</ResponseField>

</ResponseField>

</Expandable>

</ResponseField>

<RequestExample>

```bash Request
curl --location --request POST 'https://api.xente.co/v2/organizations' \
--header 'Content-Type: application/json' \
--header 'Authorization: Token <token>' \
--data-raw '{
    "name": "My company name",
    "description": "We sell widgets",
    "address": "Address of your organization",
    "user": {
    "name": "My name",
    "email": "email of user creating account",
    "phone number": "phone number of user creating account"
  }
}'
```

</RequestExample>
