---
title: "Errors"
description: "Handling Xente Error Codes"
---
The Xente API is RESTful and uses conventional HTTP response codes to indicate the success or failure of requests.

### Error codes

- **200**: Request was successful.
- **201**: A resource has successfully been created.
- **400**: A validation error occurred.
- **401**: The request was not authorized.
- **404**: Could not find the resource.
- **5xx**: Request had an error on Xente's end. If you encounter this error, please inform us immediately.
